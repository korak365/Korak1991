# Expired Domain Hunter

This scraper identifies high-authority domains that are nearing expiration by analyzing backlink profiles. It provides SEO professionals and domain traders with valuable data for acquiring expired or expiring domains.

## Features:
- Scrapes data from backlink profile sources.
- Filters domains by authority score and keywords in anchor text.
- Tracks expiration dates to identify actionable domains.

## Outputs:
- Domain name
- Authority score
- Number of backlinks
- Expiration date
- Keywords matched in anchor text
- Source backlink page URL
